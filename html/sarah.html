Télécharger LM Studio

1. Rendez-vous sur le site officiel de LM Studio : lmstudio.ai

2. Choisissez la version adaptée à votre système d’exploitation (Windows, macOS ou Linux).

3. Cliquez sur le bouton Télécharger et patientez jusqu’à la fin du téléchargement.

Installation de LM Studio

Sur Windows :
1. Ouvrez le fichier .exe téléchargé.
2. Suivez les étapes de l'assistant.
3. Acceptez et choisissez le répertoire.

Sur macOS :
1. Ouvrez le fichier .dmg téléchargé.
2. Déplacez LM Studio dans Applications.

Sur Linux :
1. Suivez la doc pour installer via terminal.

Lancer LM Studio

1. Ouvrez LM Studio depuis le menu Démarrer ou l'icône sur le bureau.

2. L'interface principale s'affichera, vous permettant de télécharger ou configurer un modèle de langage.

3.Vous pourrez télécharger et configurer des modèles de langage.
